## Role

## Short Description

I built a long-running Python/Flask application for a psychologist who needed to process large volumes of client assessment data, analyze changes over time, and generate patient-friendly outputs. The system handled data processing, longitudinal statistical analysis, chart generation, and report-ready outputs in the patient’s native language. A final formatting stage also helped turn the analytical output into polished reports suitable for real client use.

This project is a good example of code-first workflow automation for a document-heavy, high-trust use case where reliability and usability mattered as much as the analysis itself.

## Alternate Description

This project centered on a Python/Flask application for turning raw psychological test data into usable reporting workflows. It automated large-scale data handling, longitudinal analysis, visual output generation, and patient-facing report content. The broader workflow also included a final document-formatting step so the outputs were easier for the clinician to use directly with clients.
